receiver/objectify¶

module receiver.objectify¶

Objectifying message receiver.

Receives protocol.Packet instances. Converts to plain object. Passes on to next receiver. Received messages are assumed valid.

Provide protocol definition and next receiver.

class receiver.objectify.Receiver(protocol, nextReceiver)¶
Arguments:
  • protocol (protobufjs.Root) – Protocol definition.
  • nextReceiver (Receiver) – Receiver to deliver message to.
