logchan/prefix¶

module logchan.prefix¶

Add prefix Logchan.

Adds static prefix to message. Passes on to subsequent logchan.

Examples:

import ConsoleLogchan from 'cashshuffle/logchan/console'
import PrefixLogchan from 'cashshuffle/logchan/prefix'
const logConsole = new ConsoleLogchan()
const log = new PrefixLogchan('Shuffler 1: ', logConsole)

async function run () {
  await log.send('Starting run')
}

run()
class logchan.prefix.PrefixLogchan(prefix, nextLogchan)¶
Arguments:
  • prefix (string) – Prefix to add to each message.
  • nextLogchan (Logchan) – Logchan to relay messages to.
