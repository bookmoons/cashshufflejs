crypto/bitcore¶

module crypto.bitcore¶

Bitcore based implementation of message encryption.

Uses:

  • crypto for hashing.
  • bitcore-lib-cash for key operations.
  • @bookmoons/bitcore-ecies-cash for encryption operations.

Network values are bitcore-lib-cash Network instances. Standard networks are available through bitcore.Networks.
